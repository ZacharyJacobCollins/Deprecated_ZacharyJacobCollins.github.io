# zacharyjacobcollins.github.io

> Personal website, ready to run on github pages 


## Build Setup

``` bash

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build prod with minification
npm run build

# run tests (karma)
npm test
```
